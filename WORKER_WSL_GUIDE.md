# Worker WSL2 - Швидкий старт

## 🚀 Запуск

```bash
cd ~/aiwork
source venv/bin/activate
python3 worker_wsl.py
```

## 📋 Як працює:

1. **Worker запускається в WSL терміналі**
2. **Підключається до сервера** (http://34.58.51.76:8000)
3. **Бере завдання з черги**
4. **Відкриває браузер у Windows** (Edge або Chrome)
5. **Ти тестуєш у Windows з NVDA**
6. **Повертаєшся в WSL термінал**
7. **Вводиш оцінку: 1/2/3/0**
8. **Worker відправляє результат**
9. **Повторює з наступним завданням**

---

## ⌨️ Оцінки:

Після тестування введи в WSL терміналі:

- **1** = ✅ Добре (сайт доступний)
- **2** = ⚠️  Є проблеми
- **3** = ❌ Критичні помилки
- **0** = 🚫 Пропустити

## 🔧 Workflow:

1. **Запускаєш worker в WSL**
2. **Браузер відкривається у Windows**
3. **Запускаєш NVDA у Windows**
4. **Тестуєш сайт:**
   - Tab, Enter, стрілки
   - Чи читає NVDA елементи?
   - Чи логічний порядок?
   - Чи є alt-тексти?
5. **Alt+Tab назад в WSL термінал**
6. **Вводиш оцінку (1/2/3/0)**
7. **Worker відправляє результат**
8. **Відкривається наступний сайт**

---

## 💡 Поради:

### Швидше тестування:
- Тримай WSL термінал і браузер поруч
- Використовуй Alt+Tab для перемикання
- Тестуй основні речі: навігація, читання, структура

### Типові проблеми:
- **Немає alt-текстів** → оцінка 2
- **Кнопки не натискаються клавіатурою** → оцінка 3
- **Фокус не видимий** → оцінка 2
- **NVDA нічого не читає** → оцінка 3
