### 2. Worker v2 з Flask
**Файл:** `worker_v2.py`

**Як працює:**
```
Worker запускає Flask на :5000
  ↓
Відкриває 2 вкладки Chrome:
  1. Тестовий сайт (GitHub, Wikipedia...)
  2. Панель керування (localhost:5000)
  ↓
Ти тестуєш сайт з NVDA
  ↓
Переходиш на вкладку 2
  ↓
Натискаєш Alt+1/2/3/0 (або клікаєш кнопку)
  ↓
Worker отримує оцінку
  ↓
Закриває всі браузери
  ↓
Відправляє результат на сервер
  ↓
Наступне завдання
```

### 3. Панель керування
**URL:** http://localhost:5000

**Кнопки:**
- 🟢 **Alt+1** - Добре (good)
- 🟡 **Alt+2** - Проблеми (issues)
- 🔴 **Alt+3** - Критично (critical)
- ⚪ **Alt+0** - Пропустити (skipped)

**Доступність:**
- Великі кнопки
- aria-label для NVDA
- Hotkeys працюють в браузері
- Темна тема

### 4. Автоочистка
- ✅ Браузери закриваються після кожного завдання
- ✅ Завдання автоматично видаляються з черги після обробки
- ✅ Timeout 5 хв (якщо не відповів)

   - **Tab 1:** GitHub.com (тестуй тут)
   - **Tab 2:** Панель керування

3. Alt+Tab на панель
4. Натискаєш **Alt+1** (добре)
5. Chrome закривається

7. Повторюєш
