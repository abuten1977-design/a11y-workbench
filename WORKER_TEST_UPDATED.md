## 🎮 Hotkeys (Best Practice)

| Клавіша | Дія |
|---------|-----|
| **Alt+Shift+1** | ✅ Добре (доступний) |
| **Alt+Shift+2** | ⚠️ Є проблеми |
| **Alt+Shift+3** | ❌ Критичні помилки |
| **Alt+Shift+0** | 🚫 Пропустити |

**Чому Alt+Shift?**
- ✅ Стандарт для custom hotkeys в Windows
- ✅ Не конфліктує з NVDA (NVDA використовує Insert/Caps Lock)
- ✅ Не конфліктує з системою
- ✅ Легко натиснути двома руками
- ✅ Рекомендовано Microsoft для custom shortcuts

## 🐛 Troubleshooting

### Chrome не відкривається

**Перевірка:**
```bash
ls /mnt/c/Program\ Files/Google/Chrome/Application/chrome.exe
```

**Якщо немає:**
- Встанови Chrome в Windows
- Або worker відкриє дефолтний браузер

### Hotkeys не працюють

**Спробуй:**
1. Запусти worker з sudo: `sudo python3 worker.py`
2. Перевір що NVDA не перехоплює F9-F12
3. Якщо F11 конфліктує (fullscreen) - використай F10

### Worker не бачить сервер

```bash
curl http://34.58.51.76:8000/health
```

Якщо не працює - перезапусти сервер.
