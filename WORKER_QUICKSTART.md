# Швидкий старт Worker

## Крок 1: Встановити залежності (якщо ще не встановлені)

```bash
cd ~/aiwork
source venv/bin/activate
pip install playwright pynput httpx
playwright install chromium
```

## Крок 2: Запустити Worker

```bash
python3 worker.py
```

## Troubleshooting

### Помилка: "Не можу підключитись до сервера"

**Перевір що сервер працює:**
```bash
curl http://34.58.51.76:8000/health
```

**Якщо не працює:**
```bash
cd ~/aiwork
./status_server.sh
```

### Помилка: "playwright not found"

```bash
pip install playwright
playwright install chromium
```

### Помилка: "pynput not found"

```bash
pip install pynput
```

### Hotkeys не працюють

**Linux:** Можливо треба права sudo
```bash
sudo python3 worker.py
```

Або додай свого користувача в групу input:
```bash
sudo usermod -a -G input $USER
# Перелогінься
```
