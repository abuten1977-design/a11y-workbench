## TROUBLESHOOTING

### Панель не відкривається
```bash
# Перевір Flask
curl http://localhost:5000
```

### Hotkeys не працюють
- Переконайся що фокус на вкладці панелі
- Спробуй клікнути кнопку мишею
- Перевір в консолі браузера (F12) чи є помилки

### Браузери не закриваються
```bash
# Закрити вручну
taskkill.exe /F /IM chrome.exe
```

### Worker зависає
```bash
# Знайти процес
ps aux | grep worker_v2

# Завершити
kill PID
```

### Черга порожня
```bash
# Створити нові завдання
./create_test_jobs.sh
```
