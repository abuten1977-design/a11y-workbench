# Швидкий довідник правил

## ✅ Застосовуємо (6 правил)

### 1. 🔍 startcheck - Перевірка перед дією
```bash
cat file.py | head  # Подивитись що всередині
python3 -m py_compile file.py  # Перевірити синтаксис
```

### 2. 🐛 kaki - Аналіз помилок
```
Помилка → Причина → Рішення → Як запобігти
```

### 3. 📝 REPLACE_BAD_CODE - Заміна файлу
```
>3 зміни → Замінити файл повністю
```

### 4. 🤖 selfself - Автономність
```
Не питати зайвого → Шукати рішення сам
```

### 5. 📦 scriptsmany - Автоматизація
```
>3 команди → Створити .sh скрипт
```

### 6. 🎯 absolutefull - Повні шляхи
```
/home/user/file замість ./file
```

### 7. 📚 applystyle - Прикладне навчання
```
Мета → Терміни → Приклади → Градієнт
```

---

## 📋 Чеклист

Перед зміною коду:
- [ ] startcheck - перевірив?
- [ ] absolutefull - повні шляхи?
- [ ] >3 зміни? → REPLACE_BAD_CODE

Після помилки:
- [ ] kaki - проаналізував?

При автоматизації:
- [ ] >3 команди? → scriptsmany
