## 🧪 Крок 2: Тестування кожного сайту

### Завдання 1: GitHub

1. **Відкриється браузер** з GitHub
2. **Запусти NVDA:** Insert+Ctrl+N
3. **Тестуй:**
   - Натисни **Tab** кілька разів
   - Слухай що читає NVDA
   - Спробуй натиснути кнопку (Enter)
   - Перевір навігацію стрілками
4. **Оціни:**
   - Все добре? → **Ctrl+Alt+1**
   - Є проблеми? → **Ctrl+Alt+2**
   - Критично? → **Ctrl+Alt+3**

**Очікування:**
```
✅ Відмічено: ДОБРЕ (або інше)
📤 Відправка результату...
✅ Результат відправлено
💰 Заробив: $3.00
📊 Всього: 1 завдань, $3.00
```

Браузер закриється, відкриється наступний сайт.

## ✅ Що перевіряємо

### Тест 1: Різні сайти ✓
- [ ] Перший: GitHub
- [ ] Другий: Wikipedia
- [ ] Третій: StackOverflow
- **Критерій:** Кожен раз НОВИЙ сайт (не зациклюється на одному)

### Тест 2: Hotkeys працюють ✓
- [ ] Ctrl+Alt+1 закриває браузер
- [ ] Відкривається наступний сайт
- **Критерій:** Hotkey спрацьовує, браузер закривається

### Тест 3: Результати відправляються ✓
- [ ] Після hotkey: "✅ Результат відправлено"
- [ ] Показує: "💰 Заробив: $3.00"
- **Критерій:** Кожен результат відправляється на сервер

### Тест 4: Статистика правильна ✓
- [ ] Після 1 завдання: "1 завдань, $3.00"
- [ ] Після 2 завдань: "2 завдань, $6.00"
- [ ] Після 3 завдань: "3 завдань, $9.00"
- **Критерій:** Лічильник працює правильно

### Тест 5: Черга спорожніла ✓
- [ ] Після 3 завдань: "⏳ Черга порожня"
- [ ] Worker не зависає, чекає нові завдання
- **Критерій:** Worker не падає, продовжує працювати
