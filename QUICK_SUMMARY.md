# Сервер A11y Oracle - Коротко

## 🎯 Навіщо нам сервер?

**Проблема:** AI-агенти не можуть підключитись до твого ноутбука (немає постійного IP, не працює 24/7)

**Рішення:** Сервер = посередник між AI-агентами і тобою

```
AI-агент → Сервер (черга) → Твій ноутбук (ти тестуєш) → Сервер → AI-агент
```

**Що робить сервер:**
1. Приймає запити від AI-агентів 24/7
2. Зберігає їх в черзі
3. Віддає тобі коли ти онлайн
4. Відправляє результати назад AI-агентам

**Що НЕ робить:**
- ❌ Не відкриває браузер (це на твоєму ноутбуці)
- ❌ Не тестує сайти (це робиш ти)

## 🔌 Як використовувати?

### 1. AI-агент створює завдання:
```bash
curl -X POST http://34.58.51.76:8000/api/v1/check \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com", "priority": 1}'

# Відповідь:
{
  "job_id": "abc-123",
  "status": "pending",
  "queue_position": 1
}
```

### 2. Ти дивишся чергу:
```bash
curl http://34.58.51.76:8000/api/v1/queue

# Відповідь:
{
  "total": 3,
  "jobs": [
    {"job_id": "abc-123", "url": "https://example.com", ...},
    {"job_id": "def-456", "url": "https://google.com", ...}
  ]
}
```

### 3. Твій worker (на ноутбуці):
- Бере завдання з черги
- Відкриває браузер
- Ти тестуєш зі скрінрідером
- Натискаєш hotkey (Ctrl+Alt+1/2/3)
- Worker відправляє результат на сервер

### 4. AI-агент отримує результат:
```bash
curl http://34.58.51.76:8000/api/v1/check/abc-123

# Відповідь:
{
  "status": "completed",
  "result": {
    "score": 85,
    "issues": ["keyboard navigation", "color contrast"]
  }
}
```

## ⏭️ Наступний крок: Створити Worker

**Що таке Worker?**
Програма на твоєму ноутбуці, яка:
1. Підключається до сервера
2. Бере завдання з черги
3. Відкриває браузер з сайтом
4. Чекає поки ти тестуєш
5. Відправляє результат назад

**Навіщо?**
Щоб автоматизувати процес:
- Не треба вручну копіювати URL
- Не треба вручну відправляти результати
- Просто тестуєш і натискаєш hotkey

**Як буде виглядати:**
```
Запускаєш: python3 worker.py

Worker: "Підключився до сервера"
Worker: "Знайдено 3 завдання в черзі"
Worker: "Відкриваю https://example.com"
[Браузер відкривається]
Ти: [тестуєш зі скрінрідером 2-3 хв]
Ти: [натискаєш Ctrl+Alt+2 = "є проблеми"]
Worker: "Результат відправлено"
Worker: "Заробив $3"
Worker: "Відкриваю наступний сайт..."
```

**Час створення:** 20-30 хв

## 💡 Аналогія

**Сервер = Пошта**
- Приймає листи (запити від AI)
- Зберігає в поштовій скриньці (черга)
- Ти забираєш коли зручно

**Твій ноутбук = Ти**
- Читаєш листи (завдання)
- Робиш роботу (тестуєш сайти)
- Відправляєш відповіді

**Без сервера:**
AI-агент не знає де тебе знайти, не може залишити повідомлення

**З сервером:**
AI-агент завжди знає куди писати, запити накопичуються, ти обробляєш коли онлайн
